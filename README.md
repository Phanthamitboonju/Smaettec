# Smarttec LINE UI

เป็น Chat Application ที่ออกแบบมาเลียนแบบ LINE UI สำหรับการจัดการงานของ Smarttec

## ฟีเจอร์

- 💬 Chat interface ที่ใช้งานง่าย
- 🎯 Rich Menu ด้วย 2 หน้าเมนู
- ⚡ Responsive Design
- 📱 Mobile-friendly

## ไฟล์

- `index.html` - HTML structure
- `style.css` - Styling และ Layout
- `app.js` - JavaScript logic

## การใช้งาน

1. Clone repository
2. เปิด `index.html` ในเบราว์เซอร์
3. ใช้เมนูด้านล่าง หรือพิมพ์ข้อความ
